# qlik-word-cloud
Qlik Sense Word Cloud extension
This extension intends to cover the need of including word clouds into Qlik Sense

#### General Settings - Add ons
- Orientation
- Start Angle
- End Angle
- Font Max Size
- Font Min Size
- Scale
- Scale Color
- Enable Custom Range. You can Specify a specific range of colors
- From
- To
